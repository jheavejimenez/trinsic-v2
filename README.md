## Getting Started

First, run npm install

```bash
npm install
```

Second, run the development server:

```bash
 nodemon index.js
```

API routes can be accessed on [http://localhost:8080/api/endpoints](http://localhost:8080/api/endpoints). This endpoints can be edited in `routes` folder.
